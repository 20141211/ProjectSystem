1.这是一个项目管理系统，只有登陆成功的用户才能进入此项目管理系统，所以没有注册用户的功能模块，用户的新增就要由系统管理员完成，这就涉及到了新增用户的批量录入与单一录入问题。
  登陆功能：当验证码，用户名和密码匹配正确时，显示用户应用的权限。
  本系统的用户分为三个角色：程序员，项目经理，管理员。其中管理员拥有所有权限，项目经理没有人员信息查看、职务信息查看，员工工作情况查询、添加用户、添加角色的权限；程序员只有发送消息，收件信息查看，发件信息查看，客户信息查看个人信息查看，任务信息查看的权限。

表结构：用户表（包含角色一对一），权限表
